Juniper Network Traffic Analysis

Traffic Status visualization

The webpage Extensively Visualizes the traffic status with interesting graphs and charts. Each of the graphs Intuitively explains the interpretations and analysis which can be derived from the traffic data.

Source To Destination Flow of Network Traffic/Packets

The below graph gives an holistic picture of the Network Traffic. First Graph represents how the source objects and destination objects are distributed( represented via percentage) among the total traffic. The second graph represents on a whole how the source objects and destination type are distributed among the total traffic.When you hover over a source object in the first graph you can see the flow of data from the selected source to its multiple destination sources. There is an percentage distribution based on the total traffic each destination object receives. The visualization in the second graph represents the percentage distribution of the destination type for the selected Source Object.

  • Source Type

The Bar Chart represents the type of source which produces more traffic. By choosing the radio button to packets, you can see which type of source produces more packets.

Network Flow Graph

Network graph enables User to choose the Type of objects he wants to analyse. The source and destination objects are restricted to the type chosen. The edges are color coded based on the Traffic. Interesting inferences can be derived. For instance, a possibility of network bottleneck can be detected if there are multiple red or orange edges towards a node but there is only green edges going outwards, which means less traffic is going outwards.
